O QUE É D3F4ULT?

D3F4ULT é um jogo totalmente independente Point and Click, do gênero investigação, 
feito por universitários!
Ambientado em uma escola nos anos 2000, você controlará o gato azul MIKE. Passeie
pela escola e utilize o chat MSM em seu computador para descobrir a verdadeira
identidade de D3F4ULT, um colega misterioso que parece precisar muito de ajuda.

COMO JOGAR?

É muito simples! Use "A" e "D" para se mover,
Use o mouse para clicar nos objetos e interagir com eles. 
Use "ESC" para abrir o menu.
